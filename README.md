## <img src="./imgs/icon/motivation.png" width="25" /> 动机

为了帮助投稿顶会的同学进行交流，我们组织相应的顶会**交流群**供大家进行交流讨论，本仓库记录了相应**群规定和管理员信息**，为了更好，更公开的给大家提供帮助。

管理员会为大家**按时更新所有AI顶会的截稿日期（管理员会在群内提前3个月，提前1个月分别提醒）。**

在该会议公布录取结果之后，我们还组织了**上岸群**帮助大家交流论文录取后的一些问题。放到QA公开文档中。

希望所有投稿的同学都好运！每投每中！

## <img src="./imgs/icon/intro.png" width="25" /> 交流群/上岸群群规

MLNLP 顶会交流群旨在提供高质量的顶会信息同步、投稿经验交流和学术资源分享平台。扫描对应顶会交流群/上岸群管理员微信二维码，申请入群请备注「姓名 + 学校/单位 + 方向 + 顶会 + 交流/上岸」，入群后请将群昵称修改为「姓名 + 学校/单位 + 方向」，我们会定期移除没有实名的同学。

群里交流的一些【注意事项】希望大家一起维护（违反会被移除群聊）：

1. 鼓励分享顶会重要时间节点、会议动态和投稿经验等内容。
2. 禁止发布广告、营销、刷屏、无关推广、低质量灌水、引战、人身攻击或不友善言论。
3. 禁止透露任何个人投稿信息，包括投稿 ID、匿名论文标题、投稿截图及其他未公开投稿内容。
4. 禁止讨论、猜测或传播任何可能违反双盲审稿、匿名投稿和会议保密要求的信息。
5. 审稿人、AC、领域主席等相关人员不得泄露审稿内容、评分、讨论、决策或任何会议内部保密信息。
6. 未经允许，禁止公开他人论文、投稿材料、审稿意见、聊天截图或个人信息，也请勿大量添加群内成员。
7. 招聘、招生、活动和课程等信息需经管理员审核后发布；严重违反规则者将被移出群聊。
