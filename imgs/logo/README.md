# Conference Logos

The table uses year-independent logos stored locally to avoid broken external
links.

Official brand assets:

- `aaai.jpg`: AAAI organization logo from `aaai.org`.
- `iclr.svg`: ICLR logo from `iclr.cc`.
- `acl.png`: ACL organization logo from `aclweb.org`.
- `icml.svg`: ICML logo from `icml.cc`.
- `coling.png`: COLING wordmark from the ICCL website.
- `neurips.svg`: NeurIPS logo from `neurips.cc`.

Repository wordmarks:

- `naacl.svg`
- `emnlp.svg`
- `acm-mm.svg`
- `ijcai.svg`

The repository wordmarks are simple, year-independent labels used where the
official site does not provide a stable cross-year conference logo.
